# pychain

documentation at [docs\docs.md](https://github.com/OutSquareCapital/pychain/blob/master/docs/docs.md)

## Installation

```bash
uv add git+https://github.com/OutSquareCapital/pychain.git
```

## Developpement

### Setup

After cloning the repo, run:

```bash
uv sync --dev
```

### Testing

```bash
uv run tests/doctests.py
```

### Build docs

```bash
uv run scripts/build_docs.py
```
